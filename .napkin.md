## Patterns That Work
- `typing.get_type_hints(cls, include_extras=True)` resolves forward refs from `from __future__ import annotations`

- build_cog tests need 512×512 rasters — smaller ones fail is_cog check (no overviews generated)
- COGConnector materializes local .tif into a separate artifact — filter by run output, not by name exclusion

## Patterns That Don't Work
